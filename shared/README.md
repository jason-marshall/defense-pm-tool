# Shared Types and Constants

This directory contains shared TypeScript types, constants, and utilities used across both the frontend and backend.

## Structure (When Implemented)

```
shared/
├── types/          # TypeScript type definitions
├── constants/      # Shared constants
├── enums/          # Shared enumerations
└── utils/          # Shared utility functions
```

## Usage

Shared types ensure consistency between frontend and backend, reducing errors and improving developer experience.
